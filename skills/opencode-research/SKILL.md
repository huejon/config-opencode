# OpenCode Research

## Procedure

1. Record `opencode --version`.
2. Check current official docs for the feature.
3. Run a minimal local experiment.
4. Record command, directory, exit code, relevant output, confidence, and operational consequence in `~/.config/opencode-learning/`.
5. Mark uncertain behavior as experimental.

## Output contract

Return finding, evidence, re-check command, and setup consequence.
