# Finish the Job

## Procedure

1. Identify the concrete completion claim.
2. Gather missing local context with tools instead of guessing.
3. Make reversible assumptions when safe.
4. Execute the next useful local action.
5. Verify the result with command output or direct inspection.
6. If blocked, document attempts, evidence, why further progress is guesswork, and the smallest needed input.

## Stop rules

Stop only when complete and verified, blocked with evidence, at an external boundary, or waiting on a genuine user/product decision.
